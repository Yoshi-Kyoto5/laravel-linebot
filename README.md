# README
 
# Name 
Laravel-linebot


# Overview 
- 飲食店検索のLINEBot

# Description
- LINEアプリで投稿した内容を基に自動で飲食店検索し、その結果をLINE上で確認できる。
 そして、Gooleマップを開けたり、検索結果のお店に電話できる。

# Features
- オウム返しBot機能
- 飲食店の検索機能

# Requirement
- PHP | Laravel | Docker | Laradock | Git | GitHub | heroku | ngrok |
  LINE Messaging API | line-bot-sdk | Gurunavi API | Flex Message | Guzzle |
